# DIFM (Do It For Me)

Interactive AI Tax Guide Friend – a dynamic, story-driven interface built on Excalidraw and Hercules automation tools.

## Overview

This repository powers the DIFM (Do It For Me) experience: an infinitely zoomable, hand-held tax guidance canvas for the Australian taxation system. The UI blends AI-driven automation with rich visual feedback, guiding users step-by-step through their tax obligations.

Built in collaboration between Rob (design) and Bud (AI implementation), hosted at https://d-4l3rj5q1i-vercel.app/.

## Features

- 🚀 Dynamic Excalidraw canvas for interactive tax planning
- 🔍 ATO/myGov automation with user-assisted CAPTCHA
- 💬 Real-time question flow with radial UI
- 💾 Local-first data handling — secure and private
- 🌐 Live sync with Vercel for testing and preview

## Tech Stack

- **Frontend**: Excalidraw, HTML/CSS/JS
- **Automation**: Hermes Agent, Puppeteer-equivalent headless controls
- **Deployment**: Vercel
- **Visualization**: SVG, CSS glow effects, responsive embeds

## Development Workflow

To update:

1. Clone this repo locally
2. Edit canvas template or skill logic
3. Push changes — Vercel auto-deploys
4. Interact via live URL
